## Getting Started
---
Windows
```
pip install -r requirements.txt
``` 
Linux
```bash
pip3 install -r requirements.txt
``` 
### Setup
--- 
Once you have installed all of the requirements you will need to edit your config file with your discord token.
```json
config\config.json
```
